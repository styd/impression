What is this?

A JavaScript library to trigger 'impression' event whenever HTML DOM elements of your choice come into browser viewport.

When will I need it?

When you need to do something whenever an element is in viewport, something like:

  • turning on and off video or audio players based on whether user see them or not,
  • running animations once user look at them (such as simple counters above),
  • impression event tracking of specific HTML elements for your web analytics to find out the true CTR (Click-Through Rate),
  • etc.

Where can I use it?

You can use it with whatever framework you use for your app and whatever bundler you use to bundle your app because it's just plain vanilla JavaScript inside. Check out this page's source code and other examples to see how it's used in some frameworks.

Notes:

  1. Events Counter is there just to show you how bad counters can look if you start them on page load.
